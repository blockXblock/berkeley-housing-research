# Clariti vs OpenGov Research Plan

## Research Questions

### Clariti (Berkeley's Choice)
1. What cities currently use Clariti?
2. Do any have public APIs?
3. What's their data model?
4. Canadian origin - any US implementations?
5. What's their track record?

### OpenGov (SF's Choice - PermitSF)
1. PermitSF implementation details
2. SF's API architecture
3. Public access to permit data
4. Integration with DataSF
5. Lessons learned we can apply to Berkeley

### Comparison
- Feature matrix
- API capabilities
- Cost (if public)
- Implementation timeline
- User satisfaction
- Developer community

## Search Strategy
- City websites with "Clariti" or "OpenGov"
- API documentation searches
- Urban planning forums
- Government procurement records
- Academic papers using these systems
